# 🏛️ 三方辩论 (Three-Way Debate)

> 激进派 vs 保守派 vs 中立派 — Hermes Agent 原生三方辩论 skill

模拟三位立场鲜明的AI智能体就你的问题展开结构化辩论，最终产出**有明确倾向、可执行的决策建议**。

## 三方人格

| 派别 | 角色 | 核心信条 |
|------|------|---------|
| 🔴 激进派 | 破局者 | 不破不立，弯道超车 |
| 🔵 保守派 | 守成者 | 先虑败后虑胜，稳中求进 |
| ⚪ 中立派 | 仲裁者 | 兼听则明，数据说话 |

## 辩论流程

```
用户提问
  ├─ 第1轮：开篇陈词（3子智能体并行）
  ├─ 第2轮：交叉质询（3子智能体并行）
  └─ 第3轮：主持人最终裁决
```

## 安装

```bash
# 克隆到 Hermes skills 目录
git clone https://github.com/jh195845886/three-way-debate.git \
  ~/.hermes/skills/three-way-debate/
```

## 使用

在 Hermes Agent 中直接说：

```
帮我辩论一下：我应该离职创业吗？
```

或者：

```
三方视角分析：公司应该激进扩张还是稳扎稳打？
```

Agent 会自动加载 skill，并行启动 3 个子智能体进行辩论，最后给出综合结论。

## 输出示例

```
## 🏛️ 三方辩论 — 最终裁决

### 辩论摘要
| 派别 | 核心立场 | 最强论据 |
|------|---------|---------|
| 🔴 激进派 | 立刻辞职 | 自由职业市场规模年增30%，先发优势窗口期短 |
| 🔵 保守派 | 先兼职试水 | 90%自由职业者首年收入低于在职，需要安全垫 |
| ⚪ 中立派 | 6个月过渡期 | 用兼职验证市场需求，达到在职收入70%再全职 |

### 核心分歧
激进派看重机会窗口，保守派看重生存概率。

### 推荐方案
采取「6个月过渡策略」：维持现有工作的同时，每周投入15小时发展自由职业...

### 下一步行动
今晚列出3个潜在客户并发出合作邀约。
```
